Program Description

This tool allows you to quickly install and update .NET Runtime, SDK, Desktop Runtime and ASP.NET Core Runtime versions from 3.1 up to 10.0 (Preview). The program uses winget and official Microsoft installation scripts.

Application Features

32-bit Windows Application - This is a native 32-bit Windows application with built-in CLI module that provides independence and portability. The application includes an embedded winget tool, which can also be used through this app for package management.

What is .NET?

.NET is a free, cross-platform, open-source developer platform for building many different types of applications. It is maintained by Microsoft and the .NET community on GitHub.

Purpose: It is used to run and develop applications ranging from web (ASP.NET), desktop (WPF, WinForms) to cloud, mobile, and IoT services. Without the appropriate Runtime installed, applications built on this framework will not start on your system.

Core Requirements for .NET:

  • Windows 7 SP1, 8.1, 10, 11 or Windows Server equivalents.
  • Disk Space: Approx. 200MB - 1GB depending on version (SDK requires more).
  • RAM: Minimum 512MB (4GB+ recommended for development).

Opis programu

To narzędzie umożliwia szybkie instalowanie oraz aktualizowanie środowisk .NET Runtime, SDK, Desktop Runtime oraz ASP.NET Core Runtime w wersjach od 3.1 do 10.0 (Preview). Program korzysta z winget oraz oficjalnych skryptów instalacyjnych Microsoft.

Cechy aplikacji

Aplikacja 32-bitowa dla Windows - Jest to natywna 32-bitowa aplikacja Windows z wbudowanym modułem CLI, co zapewnia niezależność i przenośność. Aplikacja zawiera wbudowane narzędzie winget, które może być również używane za pośrednictwem tej aplikacji do zarządzania pakietami.

Co to jest .NET?

.NET to darmowa, wieloplatformowa platforma programistyczna o otwartym kodzie źródłowym, przeznaczona do tworzenia wielu rodzajów aplikacji. Jest rozwijana przez firmę Microsoft oraz społeczność .NET.

Do czego służy: Służy do uruchamiania i tworzenia aplikacji webowych (ASP.NET), desktopowych (WPF, WinForms), chmurowych, mobilnych oraz usług IoT.

Wymagania .NET:

  • System: Windows 7 SP1, 8.1, 10, 11 lub Windows Server.
  • Miejsce na dysku: Ok. 200MB - 1GB zależnie od wersji.
  • RAM: Minimum 512MB (zalecane 4GB+).
